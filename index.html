Three New HTML Skills

  • HTML Forms: I learned how to create forms for user input using <form> tags. These tags allow you to create interactive elements like text fields, checkboxes, and buttons, enabling user data to be submitted to a server.
  • Tables: I learned how to create structured tables using the <table> tag. Tables are useful for organizing data into rows and columns, making it easier to display and understand large datasets.
  • Audio Embedding: I learned how to embed audio files using the <audio> tag. This tag allows for audio playback directly within the webpage, with options to control playback (such as play, pause, and volume).
